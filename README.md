 **🔐 Login & Registration System (C++)**
> A beginner-friendly C++ project demonstrating authentication using file handling.
 

## 📌 Project Overview

A simple and secure **console-based authentication system** built using C++.  
This project demonstrates file handling, user authentication, and data management using C++.



## 🚀 Features
- User Registration with file storage  
- Login Authentication system  
- Forgot Password functionality  
- Data stored using file handling  
- Input validation and error handling  


## 🛠️ Technologies Used

- C++  
- File Handling (fstream)  
- Object-Oriented Programming (OOP)  


## 📂 Project Structure
- main.cpp → Controls program flow  
- login.cpp → Handles login functionality  
- register.cpp → Handles user registration  
- users.txt → Stores user data  

 
## ▶️ How to Run

1. Clone the repository  

2. Compile the code:
      ```
   g++ main.cpp -o login
      ```
   
4. Run:
     ```
   ./login
     ```
